Introduction

The highlighting for the blog is set in its .yaml configuration file and it closely matches the code in the blogdown book.

 highlightjsCDN: //cdnjs.cloudflare.com/ajax/libs
  highlightjsLang:
    - r
    - yaml
    - go
  highlightjsTheme: github
  highlightjsVersion: 9.12.0

Example 1

# Dr Seuss
a <- 1:2
fish.number <- paste(a, "fish", sep ="_")
b <- c("red", "blue")
fish.color <- paste(b, "fish", sep = "_")
print(c(fish.number, fish.color))
## [1] "1_fish"    "2_fish"    "red_fish"  "blue_fish"
